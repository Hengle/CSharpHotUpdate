Fast C Script

简介

这是一个基于C++, C#简单语法的动态解释语言。目的在于提供一个简单高效,语法灵活的动态解释语言。

它专为游戏开发而生, 是为LUA的替换方案而生。

前言

为什么要开发这个？我不喜欢LUA的语法，数据类型不全，没有内置的图形对象。

与C#或C++交互效率低下（当然也有其他的方案，如XLua, ULua, XLua）,但我感觉还是不满意。

还有Lua延迟GC的问题，容易导致内存增长，泄露。

优势

支持所有的C，C++原生数据类型, 更省内存

支持所有的C,C++操作运算符

支持utf8, utf16的字符串类型，字符串组合效率更高

支持数组, List模板

支持hash_map模板

支持hash_map模板的迭代器遍历时删除或添加容器的节点，而不影响遍历操作

精确的内存回收，没有延迟GC的开销，不会跑一段时间后卡一下的问题

支持对象，支持对象的构造函数与析构函数

支持函数内联，减少函数调用的开销

支持内置的图形对象，如Vector2, Vector3, Vector4, Panel, Ray, Matrix等

与C#的互调更安全，不用担心内存泄露的问题
